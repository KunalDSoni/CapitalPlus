# Capital Plus — Website Redesign

Modern, responsive website for **Capital Plus**, an authorized Business Correspondent of IDFC FIRST Bank, offering HR, Payroll, Staffing, and Compliance solutions.

🔗 **Live Site:** [kunaldsoni.github.io/CapitalPlus/website-redesign](https://kunaldsoni.github.io/CapitalPlus/website-redesign/)

---

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with hero, services overview, and IDFC FIRST Bank product suite |
| Products | `products.html` | Full list of 14 IDFC FIRST Bank financial products |
| Payroll | `payroll.html` | Payroll management service details |
| Staffing | `staffing.html` | Permanent & temporary staffing solutions |
| Compliance | `compliance.html` | Statutory compliance management services |
| About | `about.html` | Company story, mission, values, and team |
| Contact | `contact.html` | Contact form, office address, Google Map embed |

## Tech Stack

- **HTML5 & CSS3** — Pure static site, no frameworks
- **Inter** (Google Fonts) — Typography
- **Vanilla JavaScript** — Scroll animations, mobile nav, form handling
- **Web3Forms** — Contact form email delivery (requires access key setup)
- **GitHub Pages** — Hosting & deployment

## Run Locally

```bash
cd website-redesign
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080)
